JAPANESE VOCABULARY - First 100 Words for Beginners
    Format: romaji / pronunciation / definition / example sentence
    Goal: Conversational Japanese in 6 months! 頑張って！(Ganbare! - Do your best!)
__________________________________________________

の
    no / "noh" / possessive particle (like 's in English), also connects nouns
    私の本です。= It's my book.

に
    ni / "nee" / direction particle (to, at, in), indicates location or time
    学校に行きます。= I go to school.

は
    wa / "wah" / topic marker particle (marks what you're talking about)
    私は学生です。= I am a student.

て
    te / "teh" / verb connector particle (and, then), used to link actions
    食べて寝ます。= I eat and then sleep.

を
    wo / "oh" / object marker particle (marks the direct object of a verb)
    水を飲みます。= I drink water.

が
    ga / "gah" / subject marker particle (marks the subject, emphasizes new info)
    猫がいます。= There is a cat.

だ
    da / "dah" / casual copula (informal "is/am/are")
    これは本だ。= This is a book.

た
    ta / "tah" / past tense marker (attached to verbs)
    食べた。= I ate.

する
    suru / "soo-roo" / to do (most versatile verb - combines with nouns)
    勉強する。= I study. (lit: I do studying)

と
    to / "toh" / and (connects nouns), with, quotation marker
    犬と猫 = dog and cat

ます
    masu / "mah-soo" / polite verb ending (shows respect)
    食べます。= I eat. (polite)

で
    de / "deh" / location particle (at, in - where action happens), by means of
    家で食べます。= I eat at home.

ない
    nai / "nah-ee" / negative (not, don't), attached to verbs
    食べない。= I don't eat.

いる
    iru / "ee-roo" / to exist, to be (for living things - people, animals)
    友達がいる。= I have a friend. (lit: A friend exists)

も
    mo / "moh" / also, too (replaces は/が to mean "also")
    私も行きます。= I will go too.

ある
    aru / "ah-roo" / to exist, to have (for non-living things)
    本がある。= There is a book.

・
    [SYMBOL] nakaguro / middle dot / used to separate words, especially foreign names
    Example: コカ・コーラ (Coca-Cola)

です
    desu / "deh-soo" / polite copula (polite "is/am/are")
    私は学生です。= I am a student.

「
    [SYMBOL] opening quotation mark / used to begin quoted speech or titles

」
    [SYMBOL] closing quotation mark / used to end quoted speech or titles

こと
    koto / "koh-toh" / thing (abstract), fact, matter, nominalizer (turns verbs into nouns)
    食べることが好きです。= I like eating. (lit: I like the thing of eating)

か
    ka / "kah" / question particle (put at end to make questions)
    元気ですか？= Are you well?

なる
    naru / "nah-roo" / to become, to turn into
    大人になる。= To become an adult.

れる
    reru / "reh-roo" / passive/potential verb ending (able to, is done)
    食べられる。= Can be eaten. / Is edible.

から
    kara / "kah-rah" / from, because, since
    日本から来ました。= I came from Japan.

）
    [SYMBOL] closing parenthesis / used same as English

（
    [SYMBOL] opening parenthesis / used same as English

ん
    n / "nn" / explanatory particle (short for のです), adds nuance
    行くんです。= (It's that) I'm going.

よう
    you / "yoh" / way, manner, appearance, "like/as if"
    子供のようだ。= It's like a child.

いう
    iu / "ee-oo" / to say, to call, to be called
    何という名前ですか？= What is your name called?

う
    u / "oo" / volitional ending (let's, shall we)
    食べよう！= Let's eat!

人
    hito / "hee-toh" / person, people
    あの人は誰ですか？= Who is that person?

この
    kono / "koh-noh" / this (before nouns)
    この本は面白い。= This book is interesting.

という
    to iu / "toh-ee-oo" / called, named, that (quotative)
    田中という人 = a person called Tanaka

思う
    omou / "oh-moh-oo" / to think, to feel, to believe
    いいと思う。= I think it's good.

その
    sono / "soh-noh" / that (near listener), its
    その本をください。= Please give me that book.

的
    teki / "teh-kee" / suffix meaning "-like, -ish, -al" (makes adjectives)
    日本的 = Japanese-style

年
    nen/toshi / "nehn" / year
    今年は2025年です。= This year is 2025.

日
    hi/nichi / "hee/nee-chee" / day, sun
    今日はいい日です。= Today is a good day.

！
    [SYMBOL] exclamation mark / shows excitement or emphasis (same as English)

私
    watashi / "wah-tah-shee" / I, me (neutral, slightly formal)
    私は田中です。= I am Tanaka.

もの
    mono / "moh-noh" / thing (concrete/physical), person (affectionate)
    好きなものは何ですか？= What things do you like?

や
    ya / "yah" / and (non-exhaustive list - implies "etc.")
    りんごやみかん = apples, oranges, etc.

ば
    ba / "bah" / conditional (if), attached to verbs
    食べれば元気になる。= If you eat, you'll become healthy.

？
    [SYMBOL] question mark / optional in Japanese (か already marks questions)

それ
    sore / "soh-reh" / that (thing near listener)
    それは何ですか？= What is that?

者
    mono/sha / "moh-noh/shah" / person (formal), one who does something
    研究者 = researcher

できる
    dekiru / "deh-kee-roo" / can do, to be able to, to be possible
    日本語ができます。= I can speak Japanese.

これ
    kore / "koh-reh" / this (thing near speaker)
    これは私の本です。= This is my book.

#
    [SYMBOL] hash/number sign / used for hashtags or numbers

|
    [SYMBOL] vertical bar / separator in documents or tables

言う
    iu / "ee-oo" / to say, to tell (same as いう but kanji form)
    何と言いましたか？= What did you say?

くる
    kuru / "koo-roo" / to come
    明日来ます。= I'll come tomorrow.

月
    tsuki/getsu / "tsoo-kee/geh-tsoo" / moon, month
    今月は忙しい。= This month is busy.

として
    toshite / "toh-shee-teh" / as (in the role of)
    学生として働く。= To work as a student.

さん
    san / "sahn" / honorific suffix (Mr./Ms./Mrs.) - shows respect
    田中さんは先生です。= Mr./Ms. Tanaka is a teacher.

：
    [SYMBOL] colon / used same as in English

まで
    made / "mah-deh" / until, up to, as far as
    5時まで働きます。= I work until 5 o'clock.

たい
    tai / "tah-ee" / want to (attached to verb stems)
    食べたい！= I want to eat!

など
    nado / "nah-doh" / etc., and so on, such as
    りんごなどが好きです。= I like apples and such.

中
    naka / "nah-kah" / inside, middle, during
    家の中にいます。= I'm inside the house.

られる
    rareru / "rah-reh-roo" / potential/passive (can, able to, is done)
    見られる。= Can be seen. / Is visible.

お
    o / "oh" / honorific prefix (makes words polite)
    お水をください。= Please give me water. (polite)

ね
    ne / "neh" / sentence ending (right?, isn't it? - seeks agreement)
    いい天気ですね。= Nice weather, isn't it?

，
    [SYMBOL] comma / pause mark (Japanese also uses 、)

…
    [SYMBOL] ellipsis / indicates trailing off, hesitation, or continuation

へ
    e / "eh" / direction particle (toward) - less common than に
    日本へ行きます。= I'm going to Japan.

見る
    miru / "mee-roo" / to see, to look, to watch
    映画を見ます。= I watch a movie.

ため
    tame / "tah-meh" / for, for the sake of, in order to
    健康のために運動する。= I exercise for my health.

てる
    teru / "teh-roo" / casual contraction of ている (ongoing action)
    食べてる。= I'm eating.

そう
    sou / "soh" / that way, it seems, I heard
    そうですね。= That's right, isn't it.

日本
    nihon / "nee-hohn" / Japan
    日本は美しいです。= Japan is beautiful.

おる
    oru / "oh-roo" / to be, to exist (humble/regional form of いる)
    ここにおります。= I am here. (humble)

一
    ichi/hito / "ee-chee" / one, first
    一つください。= One, please.

ので
    node / "noh-deh" / because, since (softer than から)
    忙しいので行けません。= I can't go because I'm busy.

いく
    iku / "ee-koo" / to go
    学校に行く。= I go to school.

だけ
    dake / "dah-keh" / only, just
    水だけください。= Just water, please.

何
    nani/nan / "nah-nee" / what
    何を食べますか？= What will you eat?

よ
    yo / "yoh" / sentence ending (emphasis, assertion, new info)
    美味しいよ！= It's delicious!

自分
    jibun / "jee-boon" / oneself, myself, yourself
    自分でやります。= I'll do it myself.

しまう
    shimau / "shee-mah-oo" / to finish, to end up doing (often regret/completion)
    食べてしまった。= I ended up eating it all.

について
    ni tsuite / "nee-tsoo-ee-teh" / about, concerning, regarding
    日本について話す。= To talk about Japan.

方
    kata/hou / "kah-tah/hoh" / person (polite), way, direction
    あの方は誰ですか？= Who is that person? (polite)

時
    toki / "toh-kee" / time, when, moment
    食べる時に話さない。= Don't talk when eating.

ぬ
    nu / "noo" / classical negative ending (archaic, literary)
    知らぬ = don't know (old style)

問題
    mondai / "mohn-dah-ee" / problem, question, issue
    問題ありません。= No problem.

考える
    kangaeru / "kahn-gah-eh-roo" / to think, to consider
    よく考えます。= I'll think about it carefully.

せる
    seru / "seh-roo" / causative ending (make/let someone do)
    食べさせる。= To make/let someone eat.

また
    mata / "mah-tah" / again, also, or
    また会いましょう！= Let's meet again!

『
    [SYMBOL] opening double bracket / for titles of books/works

今
    ima / "ee-mah" / now, present time
    今、何時ですか？= What time is it now?

』
    [SYMBOL] closing double bracket / for titles of books/works

みる
    miru / "mee-roo" / to try doing (after て form), to see
    食べてみる。= I'll try eating it.

でも
    demo / "deh-moh" / but, however, even
    難しい、でも楽しい。= It's difficult, but fun.

やる
    yaru / "yah-roo" / to do (casual), to give (to inferior)
    宿題をやる。= I do homework.

わけ
    wake / "wah-keh" / reason, meaning, situation
    そういうわけです。= That's how it is.

ござる
    gozaru / "goh-zah-roo" / to be (very polite/humble - formal settings)
    ありがとうございます。= Thank you very much.

たち
    tachi / "tah-chee" / plural suffix (for people)
    友達たち = friends (group)

ところ
    tokoro / "toh-koh-roh" / place, point in time, about to
    今、食べるところです。= I'm about to eat now.

．
    [SYMBOL] period / sentence ending (Japanese also uses 。)

さ
    sa / "sah" / -ness (nominalizer for adjectives), casual filler
    大きさ = size (bigness)

いい
    ii / "ee" / good, fine, nice
    これはいいです。= This is good.

って
    tte / "tteh" / casual quotation particle (short for と)
    行くって言った。= (They) said they're going.

前
    mae / "mah-eh" / before, front, in front of, ago
    駅の前で会いましょう。= Let's meet in front of the station.

第
    dai / "dah-ee" / ordinal prefix (makes numbers into "first, second...")
    第一 = first, number one

たり
    tari / "tah-ree" / doing things like (lists non-exhaustive activities)
    食べたり飲んだりする。= I do things like eating and drinking.

性
    sei / "seh-ee" / nature, gender, -ness (suffix)
    可能性 = possibility

行く
    iku / "ee-koo" / to go (kanji form)
    日本に行きます。= I will go to Japan.

より
    yori / "yoh-ree" / than, from, rather
    犬より猫が好き。= I like cats more than dogs.

そして
    soshite / "soh-shee-teh" / and, and then, thus
    食べた。そして寝た。= I ate. And then I slept.

けど
    kedo / "keh-doh" / but, although (casual)
    好きだけど高い。= I like it, but it's expensive.

どう
    dou / "doh" / how, what way, in what manner
    どうですか？= How is it?

出る
    deru / "deh-roo" / to leave, to go out, to appear, to come out
    家を出る。= I leave the house.

時間
    jikan / "jee-kahn" / time, hour
    時間がありません。= I don't have time.

LEARNING TIPS for Conversational Japanese:

1. PARTICLES ARE KEY: は, が, を, に, で - these small words change meaning!
2. VERB ENDINGS: Learn ます (polite) vs る (casual) forms
3. PRACTICE PATTERNS: です/だ + noun, verb + ます/る
4. DAILY SENTENCES: Start with greetings and daily routines
5. LISTEN A LOT: Watch anime/dramas with Japanese subtitles

STARTER PHRASES:
• はじめまして (hajimemashite) = Nice to meet you
• よろしくお願いします (yoroshiku onegaishimasu) = Please treat me well
• すみません (sumimasen) = Excuse me / Sorry
• ありがとう (arigatou) = Thank you
• おはよう (ohayou) = Good morning
• こんにちは (konnichiwa) = Hello (daytime)
• さようなら (sayounara) = Goodbye

Keep learning! 頑張れ！(Ganbare! - You can do it!)
